## 1. Sintaxis y "Operadores" de estructura en HTML
HTML sirve para estructurar el contenido. En lugar de operadores, utiliza los siguientes sﾃｭmbolos clave:
* **Etiquetas (Tags):** Se delimitan con los sﾃｭmbolos `<` (apertura) y `>` (cierre) para definir los elementos (ej. `<h1>`, `<p>`).
* **Atributos:** Se colocan dentro de la etiqueta de apertura e igualan un nombre a un valor mediante el signo `=` y comillas (ej. `<a href="http://www.uva.es">`).

## 2. Sintaxis y Selectores en CSS
CSS se utiliza para describir la apariencia y formato de los documentos. Utiliza reglas compuestas por un **selector** y una **declaraciﾃｳn** encerrada entre llaves `{ }`. Sus sﾃｭmbolos clave son:
* **Coma `,` (Agrupaciﾃｳn):** Sirve para agrupar varios elementos a los que se les aplicarﾃ｡ el mismo estilo (ej. `H1, P, B {color: blue;}`).
* **Punto `.` (Clases):** Define o selecciona una clase que puede aplicarse a mﾃｺltiples elementos (ej. `.texto_etiqueta { background-color: blue; }`).
* **Almohadilla `#` (Identificadores):** Selecciona el identificador (`ID`) ﾃｺnico de un elemento concreto (ej. `#par1 { color: blue; }`).
* **Dos puntos `:` y Punto y coma `;`:** El `:` separa el nombre de la propiedad de su valor asignado, y el `;` marca el final de esa declaraciﾃｳn.

## 3. Operadores reales en JavaScript
Al ser un lenguaje de programaciﾃｳn con tipos de datos variables, bucles y condicionales, JavaScript sﾃｭ dispone de operadores reales:
* **Operadores Aritmﾃｩticos:**

* **Operadores de Comparaciﾃｳn:** (Devuelven `true` o `false` y se usan en condicionales `if`)

* **Operadores Lﾃｳgicos:** 
  * AND lﾃｳgico: `&&`.
